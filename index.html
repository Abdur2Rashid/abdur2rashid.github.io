AI-Powered Smart Home Assistant (Personal Project)

Radar Presence Detection · Face Recognition · Voice Automation · AWS Lambda · Amazon Alexa

Designed and developed a fully integrated AI-powered smart home assistant that combines radar-based presence detection, face recognition, voice automation, and cloud-based AI orchestration.

The system dynamically adapts home automation workflows based on who enters the environment, enabling personalized and context-aware experiences. When a recognized individual enters the space, predefined automation scenarios are triggered automatically.

The assistant integrates seamlessly with Amazon Alexa via a custom-built Alexa Skill, powered by AWS Lambda, allowing natural voice control across all connected smart devices.

Key Capabilities
  • Radar-based presence and motion detection
  • Face recognition for identity-aware automation
  • Personalized automation scenarios per recognized user
  • Voice automation via custom Amazon Alexa Skill
  • Serverless cloud architecture using AWS Lambda
  • Scalable, privacy-conscious system design

This project demonstrates my ability to combine AI, automation, and cloud technologies with real-world use cases, translating complex technical systems into intuitive user experiences.
